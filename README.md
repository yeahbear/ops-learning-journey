## 📚 学习路线概览

| 阶段 | 时间 | 核心技术 | 成果 |
|------|------|--------|------|
| **环境与基础** | 第1–2周 | Linux 命令、VM、权限管理 | 熟练使用 Ubuntu 命令行 |
| **网络与安全** | 第3–4周 | TCP/IP、防火墙、Nmap | 能排查网络问题并加固服务器 |
| **服务与脚本** | 第5–8周 | Shell、Nginx、MySQL、Python 自动化 | 实现一键部署 LEMP 环境 |
| **云原生与自动化** | 第9–12+周 | Docker、Git、Ansible、K8s、CI/CD、监控 | 构建完整 DevOps 流水线 |
